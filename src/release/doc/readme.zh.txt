                                 nnCron
                                 ~~~~~~

nnCron 是一个功能强大的计划任务管理器、提醒工具、脚本工具和自动控制管理器。nnCron 有自己的脚本语言（兼容 Forth），并允许使用 VBScript/JScript、正则表达式，而且提供大量可以扩充功能的插件。

nnCron 支持 cron 表格格式(Unix)，并使用易编辑的文本 cron 表格文件组织任务。然而，对于喜欢图形环境的用户，nnCron 提供了一个图形界面来移除、添加、编辑和执行任务、设置提醒和改变软件配置。

除了普通的计划任务功能（在预定时间启动程序、显示提醒信息和打开文档），nnCron 还具有以下功能：

  - 以系统服务形式运行任何程序
  - 以指定用户来执行任务
  - 控制和重启失败的任务和提醒
  - 在预定时间关闭、休眠计算机或使其进入睡眠状态
  - “唤醒”计算机以执行一个任务
  - 显示、隐藏、关闭、杀死、最大化、最小化和缩小指定窗口到系统托盘
  - 显示提醒信息并将其写入 Log 文件
  - 处理剪贴板/文件/注册表
  - 模拟键盘输入和鼠标动作
  - 拨号和挂断
  - 使用 PC 喇叭发出指定长度和频率的声音
  - 播放音频文件
  - 校对系统时间
  - 设置进程的优先权
  - 终止任何正在运行的进程
  - 发生严重错误后自动重启
  - 跟踪文件、特征位、窗口、进程、鼠标动作、空闲时段、键盘按键、网络的接通和断开、disk的插入、检查网络上的主机状态(ping)、计算磁盘剩余空间，等等……

nnCron 是稳定、小巧高效和易用的软件，它运行速度非常快，并提供了很多的功能可以方便的与其他 nnCron 用户协作。

系统需求，安装/卸载 nnCron
---------------------------------------
  - IBM PC 兼容机
  - Intel 奔腾处理器
  - Windows 95/98/ME/NT/2000/XP

双击下载的文件即可开始 nnCron 的安装。安装过程中需要你选择安装位置。
卸载 nCron，请通过“控制面板”里的“添加/删除程序”项。
nnCron 将其所有的配置参数保存在 nncron.ini 文件里。

---------------------------------------
运行和终止 nnCron
---------------------------------------
最方便的启动 nnCron 服务的方法是执行安装目录下的 startnncron.bat 文件。
若要终止 nnCron 的运行，你可以执行 stopnncron.bat 文件或点击托盘图标右键菜单的 'Exit' 项。
你可以在开始菜单的 nnCron 程序组里找到上面提到的 bat 批处理文件的快捷方式。
